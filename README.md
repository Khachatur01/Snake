# Snake
Classical snake game for UNIX like systems console.
___
Controls
  
    a - left
  
    s - down
    
    d - right
  
    w - up
  
    p - pause / resume
___
Run "g++ main.cpp -lncurses" to compile.
___
cmake .

make

./snake

to compile with cmake and run.

There are 2 options after selecting board size.
